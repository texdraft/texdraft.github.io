Nomlisp grammar

Lexical structure

s-expression
::=
symbol
foo, Maybe
|
qualifier s-expression
M.x, N.{1 2 3}
|
label
:x, :y
|
constant
“string”, 16r10FFFF
|
()
unit denotation
|
(s-expression…+)
parenthesized list
|
@ s-expression
@ arguments
|
⇐ s-expression
⇐ my-Ordering
|
delimited-expression
⸨x y z⸩
|
# symbol s-expression
#format “…”
|
# qualifier symbol s-expression
#Prelude.format “…”
qualifier
::=
symbol .
|
symbol . qualifier
delimited-expression
::=
opening-delimiteri s-expression… closing-delimiteri
opening-delimiteri
::=
the ith Unicode code point with Bidi_Paired_Bracket_Type = Open
closing-delimiteri
::=
the ith Unicode code point with Bidi_Paired_Bracket_Type = Close
symbol
::=
symbol-element…+
as long as constant does not generate this string
label
::=
: symbol
|
: qualifier symbol
symbol-element
::=
any character that isn't an opening/closing delimiter, whitespace, #, or \
|
\ any character

Constants

constant
::=
string
|
number
string
::=
“ double-string-element… ”
|
‘ single-string-element… ’
double-string-element
::=
any character other than \ or “
|
”
|
general-escape
single-string-element
::=
any character other than \ or ’
|
\’
|
general-escape
general-escape
::=
\\
|
\ hexadecimal-digit…+
number
::=
integer
|
rational
|
floating
integer
::=
[sign] decimal-digits
-1234
|
radix-prefix radix-digits
2r110101
rational
::=
[sign] decimal-digits / decimal-digits
10/3
|
radix-prefix [sign] radix-digits / radix-digits
13r5/4
floating
::=
floating-value
inexact floating-point
|
floating-value ♯ [width]
exact floating-point, can't be NaN or infinite
width
::=
decimal-digits
number of bits of precision
floating-value
::=
[sign] unsigned-floating [suffix]
|
[sign] NaN
IEEE 754 not a number
|
[sign] infinity
IEEE 754 infinity
suffix
::=
binary-format
|
decimal-format
binary-format
::=
b16 | b32 | b64 | b128
|
B16 | B32 | B64 | B128
decimal-format
::=
d32 | d64 | d128
|
D32 | D64 | D128
unsigned-floating
::=
radix-prefix radix-digits [. radix-digits] exponent
2r10×2↑4
|
radix-prefix radix-digits . radix-digits [exponent]
3r0.2
|
decimal-digits [. decimal-digits] decimal-exponent
4×10↑3, 0.1E+10
|
decimal-digits . decimal-digits [decimal-exponent]
0.5
exponent
::=
× radix ↑ [sign] decimal-digits
decimal-exponent
::=
exponent
|
e [sign] decimal-digits
|
E [sign] decimal-digits
sign
::=
-
|
+
radix-prefix
::=
radix r
|
radix R
radix
::=
1 | 2 | etc. | 36
radix-digits
::=
radix-digit…+
decimal-digits
::=
decimal-digit…+
decimal-digit
::=
0 | 1 | etc. | 9
hexadecimal-digit
::=
decimal-digit
|
{a | A} | {b | B} | etc. | {f | F}
radix-digit
::=
hexadecimal-digit
|
{g | G} | {h | H} | etc. | {z | Z}

Phrase structure

Basics

name
::=
symbol
|
qualified-name
qualified-name
::=
qualifier symbol

notation-template
::=
[# name] delimited-expression
parameter
::=
@ nameterm
bound to variadic argument list
|
pattern
argument
::=
expression
|
@ expression
spread variadic argument list
|
⇐ expression
instance supply to function
pattern
::=
(as nameterm pattern)
bind matched value to name
|
namepattern
nullary constructor
|
nameterm
bind any value
|
(namepattern pattern…+)
constructor pattern
|
constant
literal pattern
|
(record {label [pattern]}…+ [| nameterm])
match record with row
|
(tuple pattern…)
match tuple
|
(where pattern (pattern expression)…)
guard with additional matches

Declarations

::=

|
metadata-declaration
|
